📋 API说明

🎯 支持的功能：
  • 随机获取图片/视频文件
  • 支持指定目录获取
  • 支持分类随机获取
  • 支持分类展开/收起查看
  • 支持多目录组合查询（用逗号分隔）
  • 配置文件热重载
  • 递归子目录搜索

📂 支持的文件格式：
图片：.jpg .jpeg .png .gif .webp
视频：.mp4 .avi .wmv .mov .webm

🔗 其他端点：
  • API 文档
  • API 状态
  • tyxo的表情包库

    3D翻转 •立方体 •波浪 •分割 •螺旋 •镜像

    🎮 交互功能：
    • 点击"展开分类"查看详细分类
    • 点击分类名获取该分类随机内容
    • 点击具体目录名获取该目录随机内容
